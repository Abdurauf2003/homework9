RWD - Responsive veb-dizayn veb-sahifangizni barcha qurilmalarda yaxshi ko'rinishga olib keladi. Responsive veb-dizayn faqat HTML va CSS-dan foydalanadi. Bunga javob beruvchi veb-dizayn dastur yoki JavaScript emas. @media - CSS xossasi web-brauzer ekrani o'lchamlarini oladi va unga biriktirilgan CSS kodini o'sh o'lchamga tatbiq etadi.

GRID - CSS Grid Layout moduli satrlar va ustunlar bilan gridga asoslangan tartib tizimini taklif etadi, bu esa float va joylashishni aniqlashdan foydalanmasdan veb-sahifalarni loyihalashni osonlashtiradi. To'r tartibi bir yoki bir nechta yordamchi elementlarga ega bo'lgan asosiy elementdan iborat. FLEX-BOX - Flexbox Layout modulidan oldin to'rtta tartib rejimi mavjud edi:

  • Bloklash, veb-sahifadagi bo'limlar uchun
  • Inline, matn uchun
  • Jadval, ikki o'lchovli jadval ma'lumotlari uchun
  • Elementning aniq joylashuvi uchun joylangan
Flexible Box Layout moduli float yoki joylashishni aniqlashdan foydalanmasdan moslashuvchan moslashuvchan tartib tuzilishini loyihalashni osonlashtiradi.

animation-duration
animation-delay
animation-iteration-count
animation-direction
animation-timing-function
animation-fill-mode
animation

CSS animatsiyasidan foydalanish uchun avval animatsiya uchun ba'zi kalit kadrlarni belgilashingiz kerak. Asosiy kadrlar muayyan vaqtlarda element qanday uslublarga ega bo'lishini saqlaydi.

Bootstrap - RWD va mobil web-dasturlash uchun ishlatiladigan eng mashhur CSS freymvorki. Bootstrap 5 - Bootstrapning eng yangi versiyasi hisoblanadi.
